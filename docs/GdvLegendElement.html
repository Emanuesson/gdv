Description

GdvLegendElement Defines the small examples for GdvLayerContent- or GdvIndicators, used in a Layer to the GdvLegend. It is just an helper to provide descriptive legends.

CSS nodes

GdvLegendElement uses a single CSS node with name legendelement.
