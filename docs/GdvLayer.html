Description

GdvLayer is a object-class, that contains 2-dimensional plots of numerical data.

CSS nodes

GdvLayer uses a single CSS node with name layer.

Members

evaluate_point ()

The method used to derive the position on screen during drawing. This signal should be implemented in order to plot GdvLayerContent correctly.

  

eval_inner_point ()

Method to evaluate if a point on screen is more central with respect to the layer than another one. This is used by GdvTic to know where to plot the outside- and inside-length.
