Implemented Interfaces

GdvOnedLayer implements AtkImplementorIface and GtkBuildable.

Description

GdvOnedLayer defines a simple layer showing just a single axis. This can be combined with GdvIndicator to make an simple scale-widget

CSS nodes

GdvOnedLayer uses a single CSS node with name layeroned.
