Implemented Interfaces

GdvLayerContent implements AtkImplementorIface and GtkBuildable.

Description

GdvLayerContent is a object-class, thats intention is to basically contain all the information, that will be plotted in a single data-series of a plot.

gdv_layer_content_remove_data_point_by_index ()

gboolean
gdv_layer_content_remove_data_point_by_index
                               (GdvLayerContent *layer_content,
                                gint index);

Removes the data-point from layer_content that is stored under the index position (counted from the one added last to the one added first).

A negative index removes the point that was added first to the layer.
